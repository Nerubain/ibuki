A simple boilerplate for web projects.
- [x] Gulp 4
- [x] Babel 7
- [x] Eslint
- [x] Prettier
- [x] autoprefixer
- [x] Browsersync
- [x] Pug 
- [x] Scss

## Quick start
```bash
# install all packages
npm install

# gulp dev (watcher, browsersync)
npm run dev

# gulp build
npm run build
```
